## 🎨 Prompt 4: App Icon Design

```

Icon Specifications:
- Size: 1024x1024 pixels (iOS/Android standard)

- Style: Modern, flat design with gradients
- Must be recognizable at small sizes (60x60px)

- Gradient from deep blue (#0f3460) to purple (#764ba2)
- Or solid color with subtle texture
- Should stand out on phone screens

Main Symbol:
Choose one primary element:
Option 1: Stylized road with location pin
Option 2: Traffic light with emergency indicator
Option 3: Shield with road symbol
Option 4: Circular badge with pulse waves

Color Palette:
- Background: Blue-purple gradient
- Main icon: White or light color
- Accent: Red for emergency element
- Ensure high contrast for visibility

Design Guidelines:
- No text (icon only)
- Simple and bold shapes
- Recognizable at thumbnail size
- Unique and memorable

- Ensure it looks good next to other popular apps

Please generate:
1. Main app icon (1024x1024)
2. Small preview (60x60)
3. Alternative color scheme version
4. Dark mode variant
```

## 💡 Tips for Using These Prompts

### For Best Results:

1. **Use Prompt 1 First**: Get the perfect name
2. **Then Prompt 2**: Create the design brief
3. **Then Prompt 3**: Generate the actual logo
4. **Then Prompt 4**: Create the app icon
5. **Finally Prompt 5**: Complete brand identity

### Customization:
- Replace [brackets] with your chosen name
- Adjust colors if you prefer different schemes
- Add specific requirements for your region
- Include any cultural elements for Indore

### Iteration:
- Generate multiple versions
- Ask for variations
- Request specific changes
- Combine elements from different concepts

### File Formats to Request:
- PNG (transparent background)
- SVG (vector, scalable)
- JPG (for photos)
- PDF (for print)
- ICO (for favicon)
