# 🎯 Feature Guide - How to Use New Features

## 🚨 Emergency SOS Feature

### When to Use:
- Serious accidents with injuries
- Road blockages requiring immediate police
- Critical hazards needing urgent attention
- Any situation requiring emergency services

### How to Activate:
```
1. Click the "🚨 Emergency SOS" button (top right)
2. Emergency mode activates with RED theme
3. Form automatically sets to CRITICAL severity
4. Emergency banner appears with pulsing animation
```

### What Happens:
- ✅ Report flagged as EMERGENCY
- ✅ Sent to nearby police/ambulance
- ✅ Prioritized in authority dashboard
- ✅ Red badge on report card
- ✅ Pulsing animations for visibility

### Required Information:
- 📍 Location (exact address)
- 📞 Emergency Contact Number (required)
- 📝 Description of incident
- 📸 Photo (highly recommended)

---

## 📸 Image Upload Feature

### Why Upload Photos:
- ✅ Visual proof of incident
- ✅ Helps authorities assess severity
- ✅ Increases report credibility
- ✅ Better documentation
- ✅ Faster response time

### How to Upload:
```
1. Click "+ New Report" button
2. Fill in incident details
3. Scroll to "Upload Incident Photo"
4. Click "📷 Click to upload photo"
5. Select image from your device
6. Preview appears instantly
7. Click "Remove" if you want to change
8. Submit report
```

### Image Requirements:
- **Max Size**: 5MB
- **Formats**: JPG, PNG, GIF, WebP
- **Recommended**: Clear, well-lit photos
- **Best Practice**: Multiple angles if possible

## 🚀 Quick Start Guide

### Report Normal Incident:
```
1. Go to Reports page
2. Click "+ New Report"
3. Select type and severity
4. Enter location
5. Add description
6. Upload photo (optional)
7. Click "📤 Submit Report"
```

### Report Emergency:
```
1. Click "🚨 Emergency SOS"
2. Enter emergency contact
3. Enter location
4. Describe situation
5. Upload photo
6. Click "🚨 Send Emergency Report"
```

### View Reports:
```
1. Scroll to "Recent Reports"
2. See all reports with photos
3. Emergency reports at top
4. Click for more details
```

## 🎯 Best Practices

### When to Use Emergency SOS:
✅ **DO USE**:
- Serious accidents with injuries
- Road completely blocked
- Immediate danger to public
- Fire or explosion
- Medical emergency

❌ **DON'T USE**:
- Minor traffic jams
- Small potholes
- Routine congestion
- Non-urgent issues

### Photo Guidelines:
✅ **GOOD PHOTOS**:
- Clear and well-lit
- Shows full incident
- Multiple angles
- Close-up of damage
- Wide shot for context

❌ **AVOID**:
- Blurry images
- Too dark
- Too far away
- Irrelevant content
- Personal information visible
