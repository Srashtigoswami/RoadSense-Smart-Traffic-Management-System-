## 🚀 How to Use

### View Heatmap
1. Go to http://localhost:5173/heatmap
2. Map centers on Indore automatically
3. Toggle between "Vehicle Markers" and "Heatmap + Vehicles"
4. Click on vehicle icons to see traffic details
5. Red areas = heavy traffic, Green = smooth flow

### View Live Map
1. Go to http://localhost:5173/map
2. See real-time traffic updates
3. Watch the pulsing "Live Updates" indicator
4. Click markers for incident details
5. Markers cluster when zoomed out

### Understanding the Visualization
- **Vehicle Icons**: Each represents a traffic snapshot
- **Colors**: Match traffic severity (green = good, red = bad)
- **Heatmap**: Shows traffic density across the city
- **Clustering**: Groups nearby markers for clarity
